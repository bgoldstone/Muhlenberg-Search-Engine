<p>Typing Special Characters with Alt Codes</p>
<p>To use an Alt code, press and hold down the Alt key and type the code using the numeric key pad on the right side of your keyboard. If you do not have a numeric keypad, copy and paste the symbols from this page, or go back try another typing method.</p>
<p>Note: The same content is also available as a <a href="/media/contentassets/pdf/academics/llc/online-resources/latin/TypingGuides_printout_Latin_Windows.pdf">PDF</a>.</p>
