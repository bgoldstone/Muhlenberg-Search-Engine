  <p>At Muhlenberg, there are three main ways we augment active learning and engagement with online content: the layering of interactive presentation slides, the sharing of computer screens while working, and the collaborative annotation of common reading documents.</p>
<p>Three main collaboration tools currently in use at Muhlenberg include VoiceThread, Mirroring360, and Hypothes.is respectively.&nbsp;<span>You may use these collaboration tools as&nbsp;faculty, staff, or as&nbsp;students.&nbsp;</span></p>
<p><span>We have designed this webpage into sections based on existing user familiarity with the&nbsp;collaboration tools as well as the situation bringing you to our resources. You may use the links below to navigate to the particular section of interest or review all the content on collaboration tools.</span></p>
<p><strong><a href="#General Support Information">General Support Information</a></strong></p>
<p><strong><a href="#VoiceThread">VoiceThread</a></strong></p>
<p><strong><a href="#Hypothes.is">Hypothes.is</a></strong></p>
<p><strong><a href="#Mirroring 360">Mirroring 360</a></strong></p>
<p>&nbsp;</p>
<p><strong><a id="General Support Information"></a>General Support Information</strong></p>
<p>VoiceThread is accessed through&nbsp;the Learning Management System, Canvas.</p>
<p><span>Additionally, know that Hypothes.is support is available at&nbsp;<strong><a href="https://web.hypothes.is/help/." target="_blank">this Hypothes.is help webpage.</a>&nbsp;</strong></span></p>
<p>Hypothes.is and Mirroring 360 are used as Chrome extensions. To learn more about extensions and how to use them on various devices, do refer to this&nbsp;<strong><a href="https://support.google.com/chrome_webstore/answer/2664769?hl=en" target="_blank">Google documentation about Chrome Extensions.</a>&nbsp;</strong></p>
<p>Hypothes.is readings are housed in Canvas, while annotated <strong><a href="https://chrome.google.com/webstore/detail/hypothesis-web-pdf-annota/bjfhmglciegochdpefhhlphglcehbmek?hl=en">using the Hypothes.is Chrome extension</a>.</strong></p>
<p><strong><a href="https://chrome.google.com/webstore/detail/mirroring360-sender-for-c/iaigceaehdihlnolehbapjfbbfpnlngg" target="_blank">Mirroring 360 is also accessed via a Chrome extension</a>&nbsp;</strong>and available to use in any of Muhlenberg's techwall rooms.</p>
<p><span>For those interested in using other tools within Canvas, such as Zoom, Panopto,&nbsp; please note that our new ITMS website will ultimately include areas for the Learning Management System, Collaboration Tools, WebConferencing, and Media Services, so that each set of tools will have its own webpage accessible through the&nbsp;<strong><a href="http://www.muhlenberg.edu/ITDL" target="_blank">ITMS main website.</a></strong></span></p>
<p>&nbsp;</p>
<p><strong><a id="VoiceThread"></a>VOICETHREAD TOOL</strong></p>
<p><strong>I'm new to this (VoiceThread)</strong></p>
<p>For faculty/staff: We have the&nbsp;<strong><a href="https://docs.google.com/document/d/1zRGBipCKiabljp1tvwBhFHDDvG99R2Et_lyjxAsgyLg/edit?usp=sharing" target="_blank">following documentation regarding VoiceThread.</a>&nbsp;</strong>We also have a&nbsp;<strong><a href="https://muhlenbergcollege.hosted.panopto.com/Panopto/Pages/Viewer.aspx?id=91613942-6aba-4192-b4a1-acea003e3aaa" target="_blank">new VoiceThread video.</a></strong></p>
<p>We have the&nbsp;<a href="https://docs.google.com/document/d/1FA6WDOHMyoo0EW1maoiJYq9q6n3uqN5dEoVbEvrWP_0/edit#" target="_blank">following documentation regarding VoiceThread within Canvas.</a> Also, VoiceThread has documentation for <a href="https://voicethread.com/howto/canvas/" target="_blank"><strong>how to create and grade VoiceThread assignments</strong> </a>from within Canvas.</p>
<p>We also have instructions for<strong>&nbsp;<a href="https://docs.google.com/document/d/1ltI73aI3d4T268MG_8LXksqVARTvFwlP1racdp79Byg/edit?usp=sharing" target="_blank">captioning VoiceThreads available here.</a></strong></p>
<p><strong><span style="font-weight: 400;">If you are looking to create class groups/teams in VoiceThread activities, do refer to </span><a href="https://docs.google.com/document/d/12LXIlTkLWndRWQg7knf7XgCHPDlagkTeyXgUa-VXZuY/edit?usp=sharing">this ITMS documentation on subgroups within VoiceThread.</a></strong></p>
<p>For students: We have the following&nbsp;<a href="https://docs.google.com/document/d/1BaMp4-500m6jTlfFI_WDkYOaVSuvATpueQuBdPDR500/edit" target="_blank">student documentation for using VoiceThread</a> to complete assignments. We have a <a href="https://muhlenbergcollege.hosted.panopto.com/Panopto/Pages/Viewer.aspx?id=562e0ec7-cfbf-496b-a7d6-acea003e3aaa" target="_blank">new video on how to create a VoiceThread</a>, and a second new video on&nbsp;<a href="https://muhlenbergcollege.hosted.panopto.com/Panopto/Pages/Viewer.aspx?id=562e0ec7-cfbf-496b-a7d6-acea003e3aaa" target="_blank">how to comment on a VoiceThread activity.</a></p>
<p>To review some of the literature on using VoiceThread within courses, please visit the&nbsp;<strong><a href="https://voicethread.com/products/highered/" target="_blank">VoiceThread website here.</a></strong></p>
<p>To learn more about VoiceThread's features, do review this&nbsp;<strong><a href="https://voicethread.com/about/features/" target="_blank">website of VoiceThread's features.</a></strong></p>
<p>Additionally, the team hosted a <strong><a href="https://muhlenbergcollege.hosted.panopto.com/Panopto/Pages/Viewer.aspx?id=d9f39ffd-a76d-44d7-b186-acea003e3d0d" target="_blank">TechTuesday on the topic of VoiceThread</a>.</strong></p>
